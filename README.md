# Aether Studio - Master (Frontend)

This repository contains the frontend Aether Studio Piano application and its integrated MIDI library.

## Integrated MIDI Library
- **Preloaded Songs**: 5,087 classical works and curated pieces.
- **Format**: Optimized `.js` files in the `assets/` directory for fast loading.
- **GitHub Pages**: Hosted at `https://santaslileper.github.io/aether-studio/`.

## Connection
This frontend connects to the local **Aurum Agent** (Node.js) for search, harvesting, and advanced library management.
