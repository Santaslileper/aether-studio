# Aether Studio Piano

A premium digital music desk experience.

## 🚀 Features
- **88-Key Virtual Piano**: Fully interactive with realistic acoustics.
- **3D Sheet Music**: Dynamic sheet music rendering with paper-flip physics.
- **Instrument Library**: 40+ high-quality instrument samples.
- **MUSE GUIDANCE**: Interactive playback system for learning songs.

## 🎹 Playback
All pre-loaded songs (Rush E, Aurora, Moonlight Sonata, etc.) are available for instant playback.

Enjoy your digital music desk!
